# Getting Started with Nuxt 3

Nuxt 3 is a powerful framework built on top of Vue 3 that makes it easy to build modern web applications.

## Why Nuxt 3?

- **File-based routing** — Pages are automatically generated from your file structure.
- **Server-side rendering** — Better SEO and performance out of the box.
- **TypeScript support** — First-class TypeScript integration.
- **Auto-imports** — Components and composables are automatically imported.

## Creating a New Project

```bash
npx nuxi@latest init my-app
```

## File Structure

```
pages/          # File-based routing
components/     # Auto-imported components
composables/    # Auto-imported composables
layouts/        # Layout components
content/        # Markdown content
```

## Conclusion

Nuxt 3 is an excellent choice for building production-ready Vue applications. Give it a try!
